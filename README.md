# k-means-clustering

K-Means Clustering is (one of) the most common example of data clustering   . Data clustering is a method of unseupervised machine learning. In this method, data are separated into a predermined number (two or more) of clusters/groups based on some similarities.

K-Means Clustering aims to group data points by putting together the dimilar one and to minimize the variation within each cluster. It aims to partion n observations into k clusters in which each observation belongs to the cluster with the nearest means.
